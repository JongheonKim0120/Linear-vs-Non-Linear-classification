# Coronary heart disease classification

Course Project for "Machine Learning for Quants"
Bayes Business School, City St George's, University of London

## Project Background

This project was completed as part of the Machine Learning for Quants module. The objective was to develop and evaluate machine learning models for predicting coronary heart disease using clinical and behavioural data from a high-risk population in South Africa.

## Project Objective

The aim of this project is to classify whether a patient has coronary heart disease (`chd = 1`) or not (`chd = 0`).
