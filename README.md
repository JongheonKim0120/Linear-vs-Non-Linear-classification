# Coronary heart disease classification

## Project Background
This project was developed as coursework for the Machine Learning for Quants module in the MSc Quantitative Finance programme at Bayes Business School.

The objective of this project is to predict the presence of coronary heart disease among individuals in a high-risk region of the Western Cape, South Africa, and compare the predictive performance of linear and non-linear machine learning classifiers. 
## Dataset Description
Target Variable:
- chd

Features(9):
- sbp, tobacco, ldl, adiposity, famhist, typea, obesity, alcohol, age
